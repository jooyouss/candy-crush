# 糖果消消乐 (Candy Crush)

一个基于 HTML5 Canvas 和 JavaScript 开发的经典三消游戏。

## 🎮 在线试玩

访问 [https://jooyouss.github.io/candy-crush/](https://jooyouss.github.io/candy-crush/) 即可开始游戏！

## 🎯 游戏说明

1. 通过交换相邻的糖果，创建三个或更多相同颜色的糖果连线
2. 特殊糖果：
   - 条纹糖果：消除一整行或一整列
   - 包装糖果：消除周围的糖果
   - 炸弹糖果：消除大范围的糖果
   - 彩虹糖果：可以匹配任意颜色的糖果

## 🎨 特性

- 流畅的动画效果
- 精美的糖果图案
- 丰富的音效
- 关卡进度保存
- 响应式设计，支持移动设备

## 🛠️ 技术栈

- HTML5 Canvas
- JavaScript (ES6+)
- CSS3

## 🎵 音效

游戏包含多种音效：
- 点击音效
- 交换音效
- 消除音效
- 特殊糖果音效
- 关卡完成音效
- 背景音乐

## 📱 支持平台

- 桌面浏览器（推荐 Chrome、Firefox、Safari）
- 移动设备浏览器

## 🔄 本地开发

1. 克隆仓库：
```bash
git clone https://github.com/jooyouss/candy-crush.git
```

2. 进入项目目录：
```bash
cd candy-crush
```

3. 使用本地服务器运行（例如使用 Python 的 HTTP 服务器）：
```bash
# Python 3
python -m http.server 8000
```

4. 在浏览器中访问：
```
http://localhost:8000
```
